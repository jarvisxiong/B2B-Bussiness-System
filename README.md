# 本项目是一个B2B商城系统
从项目的架构和项目的完整实现源代码都在上边 满足 一定 高可用 高性能 高扩展性 

#搜索服务独立 solr集群 负载均衡 读写分离 负载均衡主备 DB主备 .....

还扩展了一些拓展的其他方案 其中云计算分析平台正在添加当中
